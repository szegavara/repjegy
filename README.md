# Repülőjegy Foglalási Rendszer

Ez a projekt egy egyszerű repülőjegy foglalási rendszert valósít meg Pythonban.

## Funkciók

- Jegy foglalása járatokra
- Foglalás lemondása
- Aktuális foglalások listázása

## Osztályok

- `Járat`: Absztrakt osztály a járatok alapvető attribútumaira
- `BelföldiJarat`: Belföldi járatok
- `NemzetkoziJarat`: Nemzetközi járatok
- `LégiTársaság`: Légitársaság kezelése
- `JegyFoglalás`: Jegyfoglalások kezelése

## Futtatás

Futtasd a `main.py` fájlt:

```bash
python main.py
```

## Elvárások

- Python 3.x
- Standard könyvtárak használata
